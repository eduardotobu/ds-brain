# Quick capture — 2026-06-11

Search, content recommendation, image/scene analysis, machine translation, dialogue systems, automated assistants, game playing, sciences, **machine learning is everywhere**. 
Machine learning is used for:
- Image analysis.
- Speech analysis.
- Surprass human performance in game playing.
- etc.

**Definition:**
Machine learning as a discipline it aims to design, understand and apply computer programs that learn from experience (i.e., data) for the purpose of:
- Modeling - Understanding how thing works.
- Prediction - Predict future outcomes.
- Control - Control towards a desired output configuration.

Focusing on predictions, with machine learning you can predict:
- Consumption of a product on the next week (forecasting)
- If I'm going to like a movie (recommendation sistems)
- if a molecule is going to be soluble in water 
- What is an image about? (Image recognition)
- What an english phrase means in spanish? (Translation)
So with machine learning you not only can predict events that are about to occur, **you also can predict properties you do not yet know**.
